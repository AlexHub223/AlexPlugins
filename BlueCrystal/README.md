# BlueCrystalCooking (Fixed & Clean)

**BlueCrystalCooking** is an Unturned RocketMod plugin that adds a "breaking bad" style cooking mechanic to the game. Players can use barrels to mix ingredients and create high-quality blue crystal meth, with custom effects and tools.

> [!IMPORTANT]
> This plugin was originally developed by **Ocelot** and is currently **deprecated/abandoned** by the original author.
> 
> This version has been **fixed and cleaned** by **Alex** to ensure compatibility with modern Unturned (2026+).

## 🚀 Features

- **Cooking Mechanics**: Use barrels to mix multiple ingredients into a liquid tray.
- **Processing**: Freeze liquid trays into solid blue crystals using specialized freezers.
- **Drug Effects**: Consume blue crystal bags for temporary speed and jump boosts.
- **Custom UI**: Integrated with RocketMod's translation and icon system.
- **Clean Code**: Removed all original DRM/License checks and legacy comments for better performance and readability.

## 📦 Installation

1. Download the `BlueCrystalCooking.dll`.
2. Place it into your server's `Plugins` folder.
3. Configure the item IDs and effects in `Plugins/BlueCrystalCooking/BlueCrystalCooking.configuration.xml`.

## ⚙️ Requirements

- [RocketMod](https://rocketmod.net/) for Unturned.
